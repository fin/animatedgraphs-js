API documentation

A library allowing transitions between graphs in javascript, using raphael.js as a drawing backend


Features

  • Bar chart drawing
  • Bar chart animated data changes
  • Bar chart linearly animated pivots

Missing features

  • Pie & Ring charts
  • Staged animations: framework intact; needs proper implementation
  • Element splitting&merging for more complicated pivots

              data = {'levels': ['gender','ethnicity'],
                     'data':// upper layer: datasets
